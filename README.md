# codeapha_task

 Project Overview: Random Quote Generator
This is a Random Quote Generator web app built using React + TypeScript.

It fetches quotes from the Gemenia API to display a random quote with the author's name.

If the main API fails, it uses a fallback quote to ensure the app still works.

The app includes a clean and responsive UI.

Users can click a button to generate a new quote instantly.

🔧 Features
✅ Built with React + TypeScript

🔄 Fetches quotes dynamically from Gemenia API

🔁 Uses fallback data if API fails

🎨 User-friendly and styled UI

📱 Fully responsive for mobile and desktop
